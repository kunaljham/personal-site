# kajham.com

Personal site built with [Hugo](https://gohugo.io/) and the [Bear Cub](https://github.com/clente/hugo-bearcub) theme. Deployed on [Railway](https://railway.app/).

## Local development

```sh
# Clone and install dependencies
git clone <repo-url>
cd personal-site
hugo mod tidy

# Start dev server
hugo server --buildDrafts

# Build for production
hugo
```
